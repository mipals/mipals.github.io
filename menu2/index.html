Software

  • IntegralEquations.jl - A Julia package for solving Integral Equations using the Boundary Element Method (through collocation). Includes options for viscothermal losses and the Fast Multipole Method. I intend to publish the package at the end of my PhD.

  • FMM2D.jl - A Julia interface for computing N-body interactions using the Flatiron Institute's FMM2D library.

  • SymSemiseparableMatrices.jl - A Julia package for efficient computations with a certain class of rank structured matrices.

  • GeneralizedSmoothingSplines.jl - A package for fitting smoothing splines of degrees \(2p-1\). Adds the options for restricting the shape of the splines. All underlying computations are done using SymSemiseparableMatrices.jl.

  • BEM200 - Some simple Julia code for 2D BEM computations for educational purposes.
