Hybrid Cross Approximation

In short Hybrid Cross Approximation tries to combine the algebraic approach from the Adaptive Cross Approximation with an analytical approach in order to obtain faster and more reliable algorithms. [1]

Use Chebyshrev interpolation. Approximate using ACA.

References

[1] Börm, Steffen. Fast Large-Scale Boundary Element Algorithms. 2020.
