The Sherman-Morrison Identity

The Sherman-Morrison identity deals with the inversion of a rank-\(1\) perturbed matrix. In short it states that its inverse is also a rank-\(1\) perturbed matrix. Most notably it can be used to significanly speed up the computations of the inverse of the total matrix if the inverse of rank-1 perturbed matrix is already know.

A Special Case

The special case of the Sherman-Morrison stated that the inversion of a rank-\(1\) perturbed identity matrix is also a rank-\(1\) perturbed identity matrix.

\[ (I + uv^T)^{-1} = I - \frac{uv^T}{1 + v^Tu}\ = I - v(1 + v^Tu)^{-1}v^T \]
The proof is simple. Assume that the inverse of \((I + uv^T)\) have a similar form, namely \((I + \alpha uv^T)\)

\[ (I + uv^T)(I + \alpha uv^T) = I + (uv^T + \alpha uv^T + uv^T\alpha uv^T) = I + u(1 + \alpha + v^T\alpha u)v^T. \]

Now since needs to be equal to \(I\) for all values of \(u\) and \(v\) we must have the expression inside the parenthesis must be 0 from which we can assert that

\[ \alpha = -(1 + v^Tu)^{-1}. \]

Inserting this into our assumption the formula follows.

The first expression is how the formula is usually stated. However I like the last form better since it naturally extends to the Sherman-Morrison-Woodbury formula.

The General Form

We can generalize the formula to a perturbation of a general (invertible) matrix

\[ (A + uv^T)^{-1} = A^{-1} - \frac{A^{-1}uv^TA^{-1}}{1 + v^TA^{-1}u} = A^{-1} - A^{-1}u\left(1 + v^TA^{-1}v^T\right)^{-1}v^TA^{-1}. \]
The intuition behind this is as follows: Set \(B = A + uv^T\) such that \(B^{-1} = (I + A^{-1}uv^T)^{-1}A^{-1}\). Then setting \(w = A^{-1}u\) and using the special case it follows.

Again the first expression is how the formula usually is stated, while the second is how I personally like to think of it.
