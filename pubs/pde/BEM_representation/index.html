Integral Formulations of the Helmholtz Equation

In this note we briefly explore the integral equations of problems associated with the Helmholtz equation. Theese integral equations form the basis of the Boundary Element Method (BEM).

Interior and Exterior Problems

In acoustics it is standard to distinguish between two types of problems: Namely interior and exterior problems. As the names suggest the first deals with problems inside bounded domains (objects) while the latter deals with problems on unbounded domains with objects possibly placed in them.

Direct Integral Formulation of the Interior Problem

Visually we can view an interior problem as follows

(Andersen, Peter Risby. Modelling of Acoustic Viscothermal Losses Using the Boundary Element Method: From Method to Optimization. Technical University of Denmark, 2018)

It is standard to denote the domain by \(\Omega\) and the boundary of the domain as \(\partial\Omega\). Furthermore the outward normal of the domain is denoted by \(n\). Note, however, that for non-smooth domains i.e. domains with e.g. sharp corners, the normal is not nessecarily uniquely defined.

The solution to the Helmholtz equation on the interior domain will be described as surface integral over the boundary

\[ C^{-}(x)p(x) = \int_{\partial\Omega}\left[G(x,y)\frac{\partial p}{\partial n_y} - p(y)\frac{\partial G(x,y)}{\partial n_y} \right]\ \mathrm{d}S_y, \]

where

\[ C^{-}(x) = \begin{cases} 1\ \quad &\text{if } x \text{ inside} \Omega\\ 0\ \quad &\text{if } x \text{ outside } \Omega\\ -\int_{\partial \Omega}\frac{\partial G_0(x,y)}{\partial n_y}\ \mathrm{d}S_y, \quad &\text{if }x\in\partial\Omega \end{cases}. \]

Discritzing the boundary as well as approximating the resulting integrals using a quadrature scheme, ones ends up with the following linear algebraic equations

\[ \text{diag}(\mathbf{c})\mathbf{p} = \mathbf{G}\mathbf{v} - \mathbf{F}\mathbf{p}. \]

Depending on the boundarty conditions this system can be solved in various ways.

Direct Integral Formulation of the Exterior Problem

If one still accepts a normal vector that points outwards from the interior problem when handling the exterior problem one ends with the following integral description of the solution to the Helmholtz equation.

\[ C^{+}(x)p(x) = \int_{\partial\Omega}\left[ p(y)\frac{\partial G(x,y)}{\partial n_y} - G(x,y)\frac{\partial p}{\partial n_y}\right]\ \mathrm{d}S_y, \]

where

\[ C^{+}(x) = \begin{cases} 0\ \quad &\text{if } x \text{ inside} \Omega\\ 1\ \quad &\text{if } x \text{ outside } \Omega\\ 1+\int_{\partial \Omega}\frac{\partial G_0(x,y)}{\partial n_y}\ \mathrm{d}S_y, \quad &\text{if }x\in\partial\Omega \end{cases}. \]

Similarily to the interior problem the end result here is

\[ \text{diag}(\mathbf{c})\mathbf{p} = \mathbf{F}\mathbf{p} - \mathbf{G}\mathbf{v}, \]

for which a solution also depends on the boundary conditions.
