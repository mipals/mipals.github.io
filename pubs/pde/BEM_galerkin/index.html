The Galerkin Boundary Element Approach

As an alternative to the collocation approach stands the Galerkin approach. To derive the underlying equation we start at the standard integral formulation from [1]

\[ c(\mathbf{y})p(\mathbf{y}) + \int_\Gamma\left[\frac{\partial G(\mathbf{x}, \mathbf{y})}{\partial\mathbf{n}_\mathbf{x}} - skG(\mathbf{x}, \mathbf{y})Y(\mathbf{x})\right]p(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x} = sk\int_\Gamma G(\mathbf{x},\mathbf{y})v_s(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}. \]

Multiplying the above with a test function, \(\delta(\mathbf{y})\) and integrating over the domain \(\Gamma_\mathbf{y}\) it follows that

\[ \begin{aligned} \int_\Gamma \delta(\mathbf{y}) c(\mathbf{y})p(\mathbf{y})\ \mathrm{d}\Gamma_\mathbf{y} + &\int_\Gamma \delta(\mathbf{y})\int_\Gamma\left[\frac{\partial G(\mathbf{x}, \mathbf{y})}{\partial\mathbf{n}_\mathbf{x}} - skG(\mathbf{x}, \mathbf{y})Y(\mathbf{x})\right]p(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\\ = sk&\int_\Gamma \delta(\mathbf{y})\int_\Gamma G(\mathbf{x},\mathbf{y})v_s(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}. \end{aligned} \]

The idea is now to parametrize the test function similar to the (normal derivative of) the pressure

  • \(p(\mathbf{x}) = \mathbf{T}(\mathbf{x})\mathbf{p}\),

  • \(v_s(\mathbf{x}) = \mathbf{T}(\mathbf{x})\mathbf{v_s}\),

  • \(\mathbf{\delta}(\mathbf{x}) = \mathbf{T}(\mathbf{x})\mathbf{\delta} = \mathbf{\delta}^\top\mathbf{T}(\mathbf{x})^\top\).

Inserting this into the above it follows that

\[ \begin{aligned} \int_\Gamma \mathbf{\delta}^\top\mathbf{T}(\mathbf{y})^\top c(\mathbf{y})\mathbf{T}(\mathbf{y})\mathbf{p}\ \mathrm{d}\Gamma_\mathbf{y} + &\int_\Gamma \mathbf{\delta}^\top\mathbf{T}(\mathbf{y})^\top\int_\Gamma\left[\frac{\partial G(\mathbf{x}, \mathbf{y})}{\partial\mathbf{n}_\mathbf{x}} - skG(\mathbf{x}, \mathbf{y})Y(\mathbf{x})\right]\mathbf{T}(\mathbf{x})\mathbf{p}\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\\ = sk&\int_\Gamma \mathbf{\delta}^\top\mathbf{T}(\mathbf{y})^\top\int_\Gamma G(\mathbf{x},\mathbf{y})\mathbf{T}(\mathbf{x})\mathbf{v_s}\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}. \end{aligned} \]

Now setting \(\mathbf{\delta}\) outside of parentheses

\[ \begin{aligned} \mathbf{\delta}^\top\left(\int_\Gamma \mathbf{T}(\mathbf{y})^\top c(\mathbf{y})\mathbf{T}(\mathbf{y})\ \mathrm{d}\Gamma_\mathbf{y} + \int_\Gamma \mathbf{T}(\mathbf{y})^\top\int_\Gamma\left[\frac{\partial G(\mathbf{x}, \mathbf{y})}{\partial\mathbf{n}_\mathbf{x}} - skG(\mathbf{x}, \mathbf{y})Y(\mathbf{x})\right]\mathbf{T}(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\right)&\mathbf{p}\\ =\mathbf{\delta}^\top sk\left(\int_\Gamma \mathbf{T}(\mathbf{y})^\top\int_\Gamma G(\mathbf{x},\mathbf{y})\mathbf{T}(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\right)&\mathbf{v_s}. \end{aligned} \]

The next (and final) step is to realize that the above has to hold for all \(\mathbf{\delta}\in\mathbb{C}^n\) (i.e. all arbitrary test functions). This implies that

\[ \begin{aligned} \left(\int_\Gamma \mathbf{T}(\mathbf{y})^\top c(\mathbf{y})\mathbf{T}(\mathbf{y})\ \mathrm{d}\Gamma_\mathbf{y} + \int_\Gamma \mathbf{T}(\mathbf{y})^\top\int_\Gamma\left[\frac{\partial G(\mathbf{x}, \mathbf{y})}{\partial\mathbf{n}_\mathbf{x}} - skG(\mathbf{x}, \mathbf{y})Y(\mathbf{x})\right]\mathbf{T}(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\right)&\mathbf{p}\\ = sk\left(\int_\Gamma \mathbf{T}(\mathbf{y})^\top\int_\Gamma G(\mathbf{x},\mathbf{y})\mathbf{T}(\mathbf{x})\ \mathrm{d}\Gamma_\mathbf{x}\mathrm{d}\Gamma_\mathbf{y}\right)&\mathbf{v_s}, \end{aligned} \]

which is the final system of equations that one needs to assemble and solve in the Galerkin approach to BEM. It is worth noting that if one simply chose \(\delta(\mathbf{y})\) to be delta functions, then the above reduces to the collocation approach.
