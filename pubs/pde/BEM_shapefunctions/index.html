Shape Functions

In this note we look at shape functions as a means of interpolating between nodes in a given coordinatesystem. In general the interpolation will have the form

\[ \mathbf{r}(\mathbf{\xi}) = M^TN(\mathbf{\xi}), \quad \xi \in [\mathbf{a},\mathbf{b}] \]

where \(M^T = \begin{bmatrix} \mathbf{m}_1 & \dots & \mathbf{m}_n\end{bmatrix}\) is a matrix containing the interpolation nodes, \(N(\mathbf{\xi})\) is a vector containing the shape functions and \(\mathbf{a},\mathbf{b}\in\mathbf{R}^m\) so that \([\mathbf{a},\mathbf{b}]\) describes a box in a \(m\)-dimensional space.

This note only shows shape functions for curves and surfaces. The usefulness of these shape functions comes from the fact that they can be used to approximate curves and surfaces so that approximations of curve- and surface integrals can be computed easily. The reason for this focus is that only curve and surface integrals are needed for the usage of the bounary element method in two- and three-dimensional space.

Curve integrals

As the name suggest a curve integral is an integral over a specific curve

\[ \int_{C_r} f\ \mathrm{d}\mu. \]

Now if we can parametrize the curve \(C_r\)

\[ C_r = \left\{\mathbf{r}(u)\ |\ u\in[a,b]\right\}, \]

then we can compute the integral as

\[ \int_{C_r} f\ \mathrm{d}\mu = \int_a^bf(\mathbf{r}(u))\text{Jacobi}_{\mathbf{r}}(u)\ \mathrm{d}u. \]

where

\[ \text{Jacobi}_{\mathbf{r}}(u) = \|\mathbf{r}'(u)\|_2, \]

is called the Jacobian. Informally we can view \(\text{Jacobi}_{\mathbf{r}}(u)\) as the pointwise length distortion from the \([a,b]\) space to the curve \(C_r\) caused by the parametrization \(\mathbf{r}(u)\).

Surface integrals

Similarily a surface integral is simply an integral over a surface

\[ \int_{S_r} f\ \mathrm{d}\mu. \]

If a parametrization, \(\mathbf{r}(u,v)\), of the surface can be found

\[ S_r = \left\{ \mathbf{r}(u,v), \quad u\in[a,b], v\in[c,d]\right\}, \]

then we can compute the surface integral as follows

\[ \int_{S_r} f\ \mathrm{d}\mu = \int_{c}^d\int_a^bf(\mathbf{r}(u,v))\text{Jacobi}_{\mathbf{r}}(u,v)\ \mathrm{d}u\mathrm{d}v. \]

Here the Jacobian is computed as

\[ \text{Jacobi}_{\mathbf{r}}(u) = \|\mathbf{r}_u'(u,v)\|_2\|\mathbf{r}_v(u,v)\|_2\sin(\theta(u,v)). \]

In parallel to the curve integrals we can view \(\text{Jacobi}_{\mathbf{r}}(u,v)\) as the pointwise area distortion from the \([a,b]\times [c,d]\) space to the surface \(S_r\) caused by the parametrization \(\mathbf{r}(u,v)\).

Approximations

We now briefly show how to approximate a curve integral using shape functions. The first step is the split the integral into smaller subintegrals

\[ \int_{C_r} f\ \mathrm{d}\mu = \sum_{i=1}^k\int_{\Delta C_i}f\ \mathrm{d}\mu. \]

Now each of the subintegrals can be computed using suitable parametrizations.

\[ \int_{\Delta C_i}f\ \mathrm{d}\mu = \int_{a_i}^{b_i}f(\mathbf{r}_i(u))\text{Jacobi}_{\mathbf{r}}(u)\ \mathrm{d}u. \]

The idea is now the use a shape-function for the

\[ \int_{\Delta C_i}f\ \mathrm{d}\mu \approx \int_{a_i}^{b_i}f(M_i^TN(u))\|M_i^TN'(u)\|_2\ \mathrm{d}u. \]

The integral can still, in general, be hard to compute. However in most cases it can be well approximated by a quadrature scheme. As such the integral becomes

\[ \int_{a_i}^{b_i}f(M_i^TN(u))\|M_i^TN'(u)\|_2\ \mathrm{d}u \approx \sum_{j=1}^{l} w_jf(M_i^TN(u_j))\|M_i^TN'(u_j)\|_2. \]

Substituting this back into (10) we find the curve integral can be approximated as

\[ \int_{C_r} f\ \mathrm{d}\mu \approx \sum_{i=1}^k\sum_{j=1}^{l} w_jf(M_i^TN(u_j))\|M_i^TN'(u_j)\|_2. \]

Curve shape functions

We briefly show some shape functions for curve integrals.

Linear elements

The simplest shape function is the linear shape function.

\[ N_L(u) = \begin{bmatrix} 1 - u\\ u\end{bmatrix},\quad u \in [0,1] \]

Visually we can view the elements of \(N_L(u)\) as function that are \(1\) when the other is \(0\).

Using this shape function we can interpolate linearly between \(\mathbf{m}_1\) and \(\mathbf{m}_2\) using the parametrization

\[ \mathbf{r}_L(u) = M^TN_L(u), \]

Visually this parametrization can be seen as

Now the Jacobian can we found by first computing \(N_L'(u)\)

\[ N_L'(u) = \begin{bmatrix} -1\\ 1\end{bmatrix},\quad u \in [0,1]. \]

Now we have that

\[ \text{Jacobi}_L(u) = \left\| M^TN_L'(u)\right\|_2 = \|\mathbf{m}_2 - \mathbf{m}_1\|_2 \]

Quadratic elements

Another simple shape function is the quadratic elements defined by

\[ N_Q(u) = \begin{bmatrix} \frac{1}{2}u (u - 1)\\ (1-u)(1+u)\\ \frac{1}{2}u (u + 1)\end{bmatrix}, \quad u\in[-1,1]. \]

Again we can view each element of \(N_Q(u)\) as a seperate function that is equal to \(1\) when the others are \(0\)

Using the quadratic shape functions we can interpolate between \(\mathbf{m}_1\), \(\mathbf{m}_2\) and \(\mathbf{m}_3\) using a quadratic as

\[ \mathbf{r}_Q(u) = M^TN_Q(u). \]

Visually this interpolation can be seen as

Now the Jacobian can be computed similarily to the linear shape functions. Namely that we first compute the tagent to the shape function vector

\[ \mathbf{r}_Q'(u) = \begin{bmatrix} u - \frac{1}{2}\\ -2u \\ u + \frac{1}{2}\end{bmatrix}, \]

from which we can compute the Jacobian as

\[ \text{Jacobi}_Q(u) = \left\|M^TN_Q'(u)\right\|_2 \]
