Shape Functions

In this note we look at shape functions as a means of interpolating between nodes in a given coordinatesystem. In general the interpolation will have the form

\[ \mathbf{r}(\mathbf{\xi}) = \mathbf{M}^T\mathbf{N}(\mathbf{\xi}), \quad \xi \in [\mathbf{a},\mathbf{b}] \]

where \(\mathbf{M}^T = \begin{bmatrix} \mathbf{m}_1 & \dots & \mathbf{m}_n\end{bmatrix}\) is a matrix containing the interpolation nodes (\(\mathbf{m}_i\)), \(\mathbf{N}(\mathbf{\xi})\) is a vector containing the shape functions and \(\mathbf{a},\mathbf{b}\in\mathbf{R}^m\) so that \([\mathbf{a},\mathbf{b}]\) describes a hyperretangular domain in a \(m\)-dimensional space. We will in this note only consider the cases of curves (\(m=1\)) and surfaces (\(m=2\)).

This note only shows shape functions for curves and surfaces. The usefulness of these shape functions comes from the fact that they can be used to approximate curves and surfaces so that approximations of curve- and surface integrals can be computed easily. The reason for this focus is that only curve and surface integrals are needed for the usage of the bounary element method in two- and three-dimensional space.

Curve integrals

As the name suggest a curve integral is an integral over a specific curve

\[ \int_{C_r} f\ \mathrm{d}\mu. \]

Now if we can parametrize the curve \(C_r\)

\[ C_r = \left\{\mathbf{r}(u)\ |\ u\in[a,b]\right\}, \]

then we can compute the integral as

\[ \int_{C_r} f\ \mathrm{d}\mu = \int_a^bf(\mathbf{r}(u))\text{Jacobi}_{\mathbf{r}}(u)\ \mathrm{d}u. \]

where

\[ \text{Jacobi}_{\mathbf{r}}(u) = \|\mathbf{r}'(u)\|_2, \]

is called the Jacobian. Informally we can view \(\text{Jacobi}_{\mathbf{r}}(u)\) as the pointwise length distortion from the \([a,b]\) space to the curve \(C_r\) caused by the parametrization \(\mathbf{r}(u)\).

Surface integrals

Similarily a surface integral is simply an integral over a surface

\[ \int_{S_r} f\ \mathrm{d}\mu. \]

If a parametrization, \(\mathbf{r}(u,v)\), of the surface can be found

\[ S_r = \left\{ \mathbf{r}(u,v), \quad u\in[a,b], v\in[c,d]\right\}, \]

then we can compute the surface integral as follows

\[ \int_{S_r} f\ \mathrm{d}\mu = \int_{c}^d\int_a^bf(\mathbf{r}(u,v))\text{Jacobi}_{\mathbf{r}}(u,v)\ \mathrm{d}u\mathrm{d}v. \]

Here the Jacobian is computed as

\[ \text{Jacobi}_{\mathbf{r}}(u) = \|\mathbf{r}_u'(u,v)\|_2\|\mathbf{r}_v(u,v)\|_2\sin(\mathbf{N}(\theta(u,v)). \]

In parallel to the curve integrals we can view \(\text{Jacobi}_{\mathbf{r}}(u,v)\) as the pointwise area distortion from the \([a,b]\times [c,d]\) space to the surface \(S_r\) caused by the parametrization \(\mathbf{r}(u,v)\).

Approximations

We now briefly show how to approximate a curve integral using shape functions. The first step is the split the integral into smaller subintegrals

\[ \int_{C_r} f\ \mathrm{d}\mu = \sum_{j=1}^m\int_{\Delta C_j}f\ \mathrm{d}\mu. \]

Now each of the subintegrals can be computed using suitable parametrizations.

\[ \int_{\Delta C_j}f\ \mathrm{d}\mu = \int_{a_j}^{b_j}f(\mathbf{r}_j(u))\text{Jacobi}_{\mathbf{r}}(u)\ \mathrm{d}u. \]

The idea is now the use a shape-function for the

\[ \int_{\Delta C_j}f\ \mathrm{d}\mu \approx \int_{a_j}^{b_j}f(\mathbf{M}_j^T\mathbf{N}(u))\|\mathbf{M}_j^T\mathbf{N}'(u)\|_2\ \mathrm{d}u. \]

The integral can still, in general, be hard to compute. However in most cases it can be well approximated by a quadrature scheme. Below the number of quadrature points is \(l\) (in practice \(l\approx 4\) should give a good estimate). As such the integral becomes

\[ \int_{a_j}^{b_j}f(\mathbf{M}_j^T\mathbf{N}(u))\|\mathbf{M}_j^T\mathbf{N}'(u)\|_2\ \mathrm{d}u \approx \sum_{k=1}^{l} w_kf(\mathbf{M}_j^T\mathbf{N}(u_k))\|\mathbf{M}_j^T\mathbf{N}'(u_k)\|_2. \]

Substituting this back into (10) we find the curve integral can be approximated as

\[ \int_{C_r} f\ \mathrm{d}\mu \approx \sum_{j=1}^m\sum_{k=1}^{l} w_kf(\mathbf{M}_j^T\mathbf{N}(u_k))\|\mathbf{M}_j^T\mathbf{N}'(u_k)\|_2. \]

Approximations of surface integrals follow the exact same procedure. However since the surface shape functions requires a two dimensional input the quadrature step requires a double sum resulting in \(l^2\) quadrature points. This is straight forward for the so-called quadrilateral surface shape functions but for triangular surface shape functions a trick needs to be performed.

Boundary Element Approximations

In the context of the boundary element method one is usually after integrals slightly different than a standard curve/surface integral. Namely that the function \(f(\mu)\) is not completely known. Instead it is partly known and partly unkown. What we mean is that the integrals of interest have the following form

\[ \int_{\Delta C_j} F_i(\mu) p(\mu)\ \mathrm{d}\mu, \]

where \(F_i\) is known while \(p\) is unkown. The idea is now to also use an inteporlation scheme for the \(p\) function similar to the one for the element \(\Delta C_j\). Namely that we assume that the following approxmation of \(p\) holds on element \(j\)

\[ p(\mu) \approx \mathbf{P}_j^T\mathbf{N}\left(\mathbf{r}^{-1}(\mu)\right) = \mathbf{N}\left(\mathbf{r}^{-1}(\mu)\right)^T\mathbf{P}_j, \]

where \(\mathbf{P}_j = \begin{bmatrix} p_{1}^{(j)} & \dots & p_{n}^{(j)}\end{bmatrix}^T\) is a vector containing the unknown nodal values \(p_{i}^{(j)}\) on element \(j\). Note that the introduction of \(\mathbf{r}^{-1}\) is purely for theoretical purposes. In practice we never need to compute it, as the computation of the integral require us to change the integration variables using \(\mathbf{r}(\mu)\) (so that \(\mathbf{r}^{-1}(\mathbf{r}(\mu)) = \mu\)). Using this we have that

\[\begin{aligned} \int_{\Delta C_j} F_i(\mu) \mathbf{N}\left(\mathbf{r}^{-1}(\mu)\right)\mathbf{P}_j^T\ \mathrm{d}\mu &= \int_{\Delta C_j} F_i(\mu) \mathbf{N}\left(\mathbf{r}^{-1}(\mu)\right)^T\ \mathrm{d}\mu\ \mathbf{P}_j. \end{aligned}\]

Doing this for every element we find that it is possible to approximate the full curve/surface integral as

\[ \int_{C_\mathbf{r}} F_i(\mu) p(\mu)\ \mathrm{d}\mu \approx \sum_{j=1}^n \int_{\Delta C_j} F_i(\mu) \mathbf{N}\left(\mathbf{r}^{-1}(\mu)\right)^T\ \mathrm{d}\mu\ \mathbf{P}_j = b_i. \]

Approximating further using a quadrature we have that

\[ \int_{C_r} f\ \mathrm{d}\mu \approx \sum_{j=1}^m\sum_{k=1}^{l} w_kF_i(\mathbf{M}_j^T\mathbf{N}(u_k))\|\mathbf{M}_j^T\mathbf{N}'(u_k)\|_2\mathbf{N}(u_k)^T = b_i. \]

Notice as we stated earlier that the term \(\mathbf{N}(u_k)^T\) is free of \(\mathbf{r}^{-1}\) (and \(\mathbf{r}\) for that matter). Details on how to compute \(b_i\) is left out (Just assume for the time being that it can be computed). Doing this for \(i=1,\dots, n\) a \(n\times n\) system of equations of the following form is established

\[ \left[\mathbf{F}\right]\left\{\mathbf{P}\right\} = \left\{\mathbf{b}\right\}. \]

Solving this w.r.t to our unknown nodal values \(\left\{\mathbf{P}\right\}\) we get an approximation of our unkown function \(p\).

Curve shape functions

We now introduce two types of shape functions for curve integrals.

Linear elements

The simplest shape function for curve integrals are the linear shape function.

\[ \mathbf{N}_L(u) = \begin{bmatrix} 1 - u\\ u\end{bmatrix},\quad u \in [0,1] \]

Visually we can view the elements of \(\mathbf{N}_L(u)\) as function that are \(1\) when the other is \(0\).

Using this shape function we can interpolate linearly between \(\mathbf{m}_1\) and \(\mathbf{m}_2\) using the parametrization

\[ \mathbf{r}_L(u) = \mathbf{M}^T\mathbf{N}_L(u), \]

Visually this parametrization can be seen as

Now the Jacobian can we found by first computing \(\mathbf{N}_L'(u)\)

\[ \mathbf{N}_L'(u) = \begin{bmatrix} -1\\ 1\end{bmatrix},\quad u \in [0,1]. \]

Now we have that

\[ \text{Jacobi}_L(u) = \left\| \mathbf{M}^T\mathbf{N}_L'(u)\right\|_2 = \|\mathbf{m}_2 - \mathbf{m}_1\|_2 \]

Quadratic elements

Another simple shape function is the quadratic elements defined by

\[ \mathbf{N}_Q(u) = \begin{bmatrix} \frac{1}{2}u (u - 1)\\ (1-u)(1+u)\\ \frac{1}{2}u (u + 1)\end{bmatrix}, \quad u\in[-1,1]. \]

Again we can view each element of \(\mathbf{N}_Q(u)\) as a seperate function that is equal to \(1\) when the others are \(0\)

Using the quadratic shape functions we can interpolate between \(\mathbf{m}_1\), \(\mathbf{m}_2\) and \(\mathbf{m}_3\) using a quadratic as

\[ \mathbf{r}_Q(u) = \mathbf{M}^T\mathbf{N}_Q(u). \]

Visually this interpolation can be seen as

Now the Jacobian can be computed similarily to the linear shape functions. Namely that we first compute the tagent to the shape function vector

\[ \mathbf{r}_Q'(u) = \begin{bmatrix} u - \frac{1}{2}\\ -2u \\ u + \frac{1}{2}\end{bmatrix}, \]

from which we can compute the Jacobian as

\[ \text{Jacobi}_Q(u) = \left\|\mathbf{M}^T\mathbf{N}_Q'(u)\right\|_2 \]

Surface Shape Functions

The idea behind the surface shape functions are the exact same as for the curve shape functions. The only thing is that our parameterization now have a two-dimensional input.

Quadrilateral Shape Functions

We start by introducing the quadrilateral surface shape functions. The main reason to use this specific kind of shape function is because that they are easy to integrate, as they are defined on a rectangular domain. However the downside of the quadrilateral surface shape functions is that it is hard to efficiently create the underlying mesh.

Linear

The linear surface shape functions can be described by

\[ \mathbf{N}_L(u,v) = \begin{bmatrix} \frac{1}{4}\left(1-u\right)\left(1-v\right)\\ \frac{1}{4}\left(1+u\right)\left(1-v\right)\\ \frac{1}{4}\left(1+u\right)\left(1+v\right)\\ \frac{1}{4}\left(1-u\right)\left(1+v\right) \end{bmatrix}, \quad u,v\in[-1,1]. \]

Quadratic

The quadratic shape functions can be described by

\[ \mathbf{N}_Q(u,v) = \begin{bmatrix} \frac{1}{4}\left(1-u\right)\left(v-1\right)\left(u+v+1\right)\\ \frac{1}{4}\left(1+u\right)\left(v-1\right)\left(v-u+1\right)\\ \frac{1}{4}\left(1+u\right)\left(1+v\right)\left(u+v-1\right)\\ \frac{1}{4}\left(u-1\right)\left(1+v\right)\left(u-v+1\right)\\ \frac{1}{2}\left(1-v\right)\left(1-u^2\right)\\ \frac{1}{2}\left(1+u\right)\left(1-v^2\right)\\ \frac{1}{2}\left(1+v\right)\left(1-u^2\right)\\ \frac{1}{2}\left(1-u\right)\left(1-v^2\right) \end{bmatrix}, \quad u,v\in[-1,1]. \]

Again we can visualize every element by itself.

Triangular Shape Functions

We now briefly introduce triangular surface shape functions. In constrast to the quadrilateral surface shape functions there exist many efficient ways of computing these triangular meshes. However the integration of these shape functions requires an extra step, since they are defined on a triangular domain instead of the usual rectangular domain.

Linear

The linear triangular surface shape functions are defined as follows

\[ \mathbf{N}_L(u,v) = \begin{bmatrix}1-u-v\\u\\v\end{bmatrix}, \quad u\in[0,1],v\in[0,1-u]. \]

Quadrature On Triangular Shape Functions

In order for us to integrate over the triangular domains using a quadrature scheme we need to transform the coordinate into a rectangular domain. This can be done with teh following transformation

\[ \begin{bmatrix}u \\ v\end{bmatrix} = \begin{bmatrix} \xi\\ \eta(1-\xi)\end{bmatrix}, \xi,\eta \in [0,1]. \]

The Jacobian in this case can be computed analytically as

\[ \text{Jacobi}_T(\xi,\eta) = \left|\det\left(\begin{bmatrix}1 & 0\\ -\eta & 1-\xi\end{bmatrix}\right)\right| = 1-\xi. \]

Note that since \(\xi \in [0,1]\) we can remove the absolute value.

\[\begin{aligned} \int_0^{1-u}\int_0^1 &f(r(u,v))\text{Jacobi}_r(u,v)\ \mathrm{d}u\mathrm{d}v = \\ &\int_0^1\int_0^1 f(r(\xi,\eta(1-\xi))\text{Jacobi}_r(\xi,\eta(1-\xi))\text{Jacobi}_T(\xi,\eta) \ \mathrm{d}\xi\mathrm{d}\eta = \\ &\int_0^1\int_0^1 f(r(\xi,\eta(1-\xi))\text{Jacobi}_r(\xi,\eta(1-\xi))(1 - \xi)\ \mathrm{d}\xi\mathrm{d}\eta. \end{aligned}\]

Given that the above integral is now defined on a rectangular domain we can approximate the integral using a suitable quadrature scheme in both the \(\xi\) and \(\eta\) direction.

We therefore end up with

\[\begin{aligned} \int_0^{1-u}\int_0^1 &f(r(u,v))\text{Jacobi}_r(u,v)\ \mathrm{d}u\mathrm{d}v = \\ &\sum_{i=1}^{l_1}\sum_{j=1}^{l_2} w_iw_jf(r(\xi_i,\eta_j(1-\xi_i))\text{Jacobi}_r(\xi_i,\eta_j(1-\xi_i))(1 - \xi_i) \end{aligned}\]
