The Boundary Element Method (BEM)

Discretization

\[ \mathbf{C}\mathbf{p} = \mathbf{H}\mathbf{p} - \mathbf{B}\frac{\partial \mathbf{p}}{\partial n} + \mathbf{p}^I = 0 \] \[ \mathbf{A}\mathbf{p} - \mathbf{B}\frac{\partial \mathbf{p}}{\partial n} + \mathbf{p}^I = 0 \]

BEM with losses

\[ \mathbf{X}\mathbf{p}_a = \mathbf{Y}_1 \mathbf{v}_{\text{boundary},n} + \mathbf{Y}_2\mathbf{v}_{\text{boundary},t} + \mathbf{Y}_3\mathbf{p}^I \] \[\begin{aligned} \mathbf{A}_a\mathbf{p}_a\frac{\partial \mathbf{p}_a}{\partial n} + \mathbf{p} &= \mathbf{0},\\ \mathbf{A}_h\mathbf{p}_h\frac{\partial \mathbf{p}_h}{\partial n} + \mathbf{p} &= \mathbf{0},\\ \mathbf{A}_v\mathbf{p}_v\frac{\partial \mathbf{p}_v}{\partial n} + \mathbf{p} &= \mathbf{0}, \quad \text{with} \quad \nabla\cdot \mathbf{v}_v = 0 \end{aligned}\]
