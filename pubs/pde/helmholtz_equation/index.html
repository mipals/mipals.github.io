The Helmholtz Equation

The Helmholtz equation can in short be viewed as the eigenvalue problem for the Laplace oeprator. It corresponds to the linear partial differential equation

\[ \Delta p + k^2p = 0, \]

where \(\Delta\) is the Laplace operator, \(k^2\) the eigenvalue and \(p\) the corresponding eigenfunction. When applied to waves the number \(k\) is known as the wave number.

The fundamental solution (Green's function)

The inhomogeneous Helmholtz equation have the following form

\[ \Delta p(x) + k^2p(x) = -f(x), \quad x\in\mathbb{R}^n, \]

where \(f:\mathbb{R}^n \rightarrow \mathbb{C}\) has compact support and \(n=1,2,3\).

For uniqueness purposes we need to specify boundary conditions at infinity. Typically the Sommerfeld radition condition is imposed

\[ \lim_{r\rightarrow \infty} r^{\frac{n-1}{2}}\left[\frac{\partial}{\partial r} - ik\right]p(r\tilde{x}) = 0, \]

uniformly in \(\tilde{x}\) with \(\|\tilde{x}\|_2 = 1\).

With this condition the solution to the Helmholtz equation is given by a convolution

\[ p(x) = \left(G * f\right)(x) = \int_{R^n}G(x-y)f(y)\ \mathrm{d}y. \]

Note that the integral is only over a finite region as we have assumed that \(f\) has compact support. The function \(G(x)\) is the so-called Green's function of the Helmholtz equation. In short this means that \(G(x)\) solves the following equation

\[ \Delta G(x,y) + k^2G(x,y) + \delta(x-y) = 0, \]

where \(\delta(x - y)\) is the Dirac delta function.

Green's function in three dimensions

In three dimensions the Green's function is

\[ G_3(x,y) = \frac{\mathrm{e}^{-ik\|x-y\|_2}}{4\pi\|x-y\|_2}. \]

Gradient of the Green's function

The computation of the Gradient is again based on the application of the identity

\[ \nabla(f \circ g) = (f' \circ g)\nabla g, \]

together with the standard product rule. We start be simply looking at when \(y=0\) resulting in

\[\begin{aligned} \nabla_x G_3(x) &= \frac{\mathrm{e}^{-ik\|x\|_2}}{4\pi}\left[\nabla\left((x^\top x)^{-1/2}\right)\right] + \frac{1}{4\pi\|x\|_2}\left[\nabla\left(\mathrm{e}^{-ik\|x\|_2}\right) \right]\\ &= \frac{\mathrm{e}^{-ik\|x\|_2}}{4\pi}\left[-\frac{1}{2}\left(x^\top x\right)^{-3/2}\nabla(x^\top x)\right] + \frac{1}{4\pi\|x\|_2}\left[-ik\mathrm{e}^{-ik\|x\|_2}\nabla\left(\|x\|_2\right)\right]\\ &= -\frac{\mathrm{e}^{-ik\|x\|_2}}{4\pi\|x\|_2^{3}}x + \frac{-ik\mathrm{e}^{-ik\|x\|_2}}{4\pi\|x\|_2^{2}}x\\ &= -\frac{\mathrm{e}^{-ik\|x\|_2}\left(ik\|x\|_2 + 1\right)}{4\pi \|x\|_2^3}\begin{bmatrix}x_1\\x_2\\x_3\end{bmatrix}. \end{aligned}\]

Now if we want the derivative with respect to \(y\neq 0\) then it follows that

\[\begin{aligned} \nabla_y G_3(x,y) &= -\frac{\mathrm{e}^{-ik\|x - y\|_2}\left(ik\|x - y\|_2 + 1\right)}{4\pi \|x\|_2^3}\begin{bmatrix}y_1 - x_1\\y_2 - x_2\\y_3 - x_3\end{bmatrix}. \end{aligned}\]

Which result in the following directional derivative

\[\begin{aligned} \frac{\partial G(x,y)}{\partial n_y} &= \frac{-\mathrm{e}^{-ik\|x\|_2}\left(ik\|x\|_2 + 1\right)}{4\pi \|x\|_2^3}(y - x)^\top n. \end{aligned}\]

Further notes

If \(k=0\) then the Helmholtz equation reduces to the Lapalce's equation

\[ \Delta_y G(x,y) = -\delta (x - y). \]

The solution to this is called the elementary solution as is equal to

\[ G_0(x,y) = \frac{1}{4\pi\|x - y\|_2}. \]

While seemlingly not relevant to the above problem it is used when computing the so-called integral free term in the boundary element method.
