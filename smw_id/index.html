The Sherman-Morrison-Woodbury identity

The Sherman-Morrison-Woodbury identity is the generalization of the Sherman-Morrison identity. In short it says that the inverse of a rank-\(k\) perturbed matrix is also a rank-\(k\) perturbed matrix.

A Special Case

A special case of the Sherman-Morrison-Woodbury identity is the following

\[ (I + UV^T)^{-1} = I - U(I + V^TU)^{-1}V^T. \]

The derivation is simple. Assume the inverse have the form \(I + UZV^T\), then we have that

\[\begin{aligned} I &= (I + UV^T)(I + UZV^T)\\ & = I + (UV^T + UZV^T + UV^TUZV^T)\\ &= I + U(I + Z + V^TUZ)V^T. \end{aligned}\]

From the fact the most right expression needs to be equal to \(I\) we must have that

\[ Z = (I + V^TU)^{-1}(-I) = -(I + V^TU)^{-1}, \]

which confirms the identity.

I sometimes find it easier to think of this identity in terms of visuals

From this we can see that the identity can be used to reduce the problem of inverting a \(n\times n\) matrix to inverting a \(k\times k\) matrix at the expense of some matrix-vector products. If \(k \ll n\) then this reduction from \(O(n^2)\) to \(O(k^3 + kn)\) flops is a really good improvement.

The General Form

In general the Sherman-Morrison-Woodbury identity have the form

\[ (A + USV^T)^{-1} = A^{-1} - A^{-1}U(S^{-1} + V^TA^{-1}U)^{-1}V^TA^{-1}. \]

The derivation here is to set \(B = A + USV^T\) so that \(B^{-1} = (I + A^{-1}USV^T)^{-1}A^{-1}\). Now set \(W = A^{-1}U\) and \(Z = SV\) and use the special form from the previous section.

An alternatively one can use a similar approach to the one presented in the previous section. The guess just has to be modified to the form \(A^{-1} + A^{-1}UZV^TA^{-1}\). What is then left is to multiply with \((A + USV^T)\) and isolate \(Z\).

Also in this case i like to think of the formula in terms of visuals

Here we only get a speed-up in computation time if \(A^{-1}\) (or simply a factorization of \(A\)) is already computed.

Code

The Julia library WoodburyMatrices.jl can be used to effeciently perform computations with Woodbury matrices.

using WoodburyMatrices, LinearAlgebra
n = 5000;
k = 10;
A = randn(n,n) + I;
U = randn(n,k);
C = randn(k,k)
V = randn(n,k); 
Woodbury_struct = Woodbury(A, U, C, V');
Woodbury_dense = A + U*C*V';
@time Woodbury_struct\ones(n);
@time Woodbury_dense\ones(n);
  0.970577 seconds (10 allocations: 190.850 MiB)
  0.963374 seconds (8 allocations: 190.850 MiB)

As expected we do not get a speed up as the inversion of the Sherman-Morrison-Woodbury matrix also requires us to invert a \(n\times n\) matrix. Thankfully the package supports a factorization of \(A\) instead of \(A\) directly. Using this a clear speed-up can be seen. Off course it is not a completely fair comparison as we have not included the time it takes to actually factorize \(A\).

n = 5000;
k = 10;
A = randn(n,n) + I;
U = randn(n,k);
C = randn(k,k);
V = randn(n,k);
Woodbury_struct = Woodbury(lu(A), U, C, V');
Woodbury_dense = A + U*C*V';
@time Woodbury_struct\ones(n);
@time Woodbury_dense\ones(n);
  0.028874 seconds (6 allocations: 78.375 KiB)
  0.953829 seconds (8 allocations: 190.850 MiB)
